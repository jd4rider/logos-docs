# Support Logos AI

Logos AI is free to start, but support helps fund development, releases, docs, voice setup work, and the time it
takes to keep the web app, desktop app, TUI, and CLI improving together.

## The three support paths

<CardGrid stagger>
  <Card title="GitHub Sponsors" icon="heart">
    Best fit for recurring support and for people who want to back the open-source side of the project directly from GitHub.
  </Card>
  <Card title="Buy Me a Coffee" icon="coffee">
    Best fit for quick one-time support, lighter recurring support, and normal users who want the fastest path.
  </Card>
  <Card title="Paid Setup Help" icon="seti:tools">

## What setup help can cover

- install the desktop app, CLI, or TUI on macOS, Windows, or Linux
- set up Ollama for local AI commentary and Logos Chat
- install Kokoro or Piper for better read-aloud voices
- guide API.Bible app creation and key placement
- help you choose a simple path between free, BYOK, and future managed options

For direct setup help, contact:

- `jd4rider@gmail.com`
- `406-396-7246`

## Important note

These links should be treated as project support, not as tax-deductible charitable donations.

If you just want to get started first, keep using the free docs and install paths:

```bash
brew tap jd4rider/logos-ai https://github.com/jd4rider/homebrew-logos-ai
brew install --cask logos-ai
```

```powershell
scoop bucket add logos-ai https://github.com/jd4rider/scoop-logos-ai
scoop install logos-ai
```

```bash
curl -fsSL https://raw.githubusercontent.com/jd4rider/logos-releases/main/install.sh | bash
```
